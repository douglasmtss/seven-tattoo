# FAQ — Perguntas Frequentes

### 1. Como faço login na área administrativa?
Acesse `/admin/login` e entre com suas credenciais. Apenas usuários autorizados têm acesso.

### 2. Como adicionar uma nova foto na galeria?
Após login, vá em "Galeria" no menu lateral, clique em "Adicionar" e preencha os campos necessários.

### 3. Como responder mensagens de contato?
No painel admin, acesse "Mensagens". Você pode visualizar, marcar como lida e copiar o e-mail para responder.

### 4. Como editar informações do estúdio?
Acesse "Configurações" ou "Sobre Nós" no menu admin e altere os campos desejados. As alterações são salvas automaticamente.

### 5. Como visualizar estatísticas de acesso?
No menu admin, clique em "Analytics" para ver gráficos de visitas e interações.

### 6. Como faço o deploy do projeto?
O deploy recomendado é na Vercel. Basta conectar o repositório e seguir as instruções da plataforma.

### 7. Onde ficam salvos os dados?
Todos os dados (galeria, mensagens, agendamentos, configurações) ficam em arquivos JSON na pasta `data/`.

### 8. Como restaurar dados ou fazer backup?
Basta copiar os arquivos da pasta `data/` para backup ou restaurar.

### 9. Como rodar os testes?
Execute `npm test` no terminal.

### 10. Como atualizar dependências?
Execute `npm update` ou edite o `package.json` e rode `npm install`.
